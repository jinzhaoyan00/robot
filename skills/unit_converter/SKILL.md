# 单位换算技能

对常见物理量进行单位换算，支持长度、重量和温度三大类别。

## 适用场景

- 用户询问"100公里等于多少英里"
- 用户询问"50千克换算成磅是多少"
- 用户询问"37摄氏度是多少华氏度"
- 任何涉及不同单位之间换算的请求

## 支持的单位

### 长度

| 标准键  | 接受的中英文别名               |
|---------|-------------------------------|
| km      | 千米、公里                    |
| mile    | 英里                          |
| m       | 米                            |
| ft      | 英尺、尺                      |
| cm      | 厘米、公分                    |
| inch    | 英寸、寸                      |

### 重量

| 标准键  | 接受的中英文别名               |
|---------|-------------------------------|
| kg      | 千克、公斤                    |
| lb      | 磅                            |
| g       | 克                            |
| oz      | 盎司                          |

### 温度

| 标准键      | 接受的中英文别名               |
|-------------|-------------------------------|
| celsius     | 摄氏度、摄氏、℃              |
| fahrenheit  | 华氏度、华氏、℉              |

## 参数说明

| 参数       | 类型   | 说明              |
|------------|--------|-------------------|
| value      | number | 待换算的数值      |
| from_unit  | string | 原单位（中英文均可）|
| to_unit    | string | 目标单位（中英文均可）|

## 示例

```json
// 100 千米换算成英里
{"skill": "unit_converter", "value": 100, "from_unit": "千米", "to_unit": "英里"}

// 50 千克换算成磅
{"skill": "unit_converter", "value": 50, "from_unit": "kg", "to_unit": "lb"}

// 37 摄氏度换算成华氏度
{"skill": "unit_converter", "value": 37, "from_unit": "摄氏度", "to_unit": "华氏度"}
```

## 返回值

格式化的换算结果字符串，例如：`100 千米 = 62.1371 英里`

单位不受支持时返回错误说明及可用换算列表。
